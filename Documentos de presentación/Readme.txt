
La tecnología cliente servidor se refiere a un modelo de comunicación en la cual, el cliente realiza cierta solicitud al servidor y este procesa la petición y retorna la respuesta.

Esta tecnología se explica un poco mejor con TCP/IP, el cual es un conjunto de protocolos de comunicación que se utilizan en la red, en este protocolo el cliente y el servidor se comunican mediante la red utilizando sus debidas direcciones IP y sus puertos.
Front-end se refiere a la parte más visual de la aplicación, ósea lo que interactúa directamente con el usuario, cómo interfaces de usuario.
Back-end se representa como la parte más interna del sistema, la cual se encarga del procesamiento de los datos, cómo bases de datos, procesos de información, reglas y/o lógicas de negocio.

Resumiendo, la tecnología cliente servidor permite que el cliente realice solicitudes a un servidor a través de la red utilizando TCP/IP, comunicando el cliente y el servidor mediante sus debidas IP y puertos,
mientras que Front-end se refiere a lo que interactúa directamente con el cliente o usuario, el Back-end se encarga del funcionamiento del sistema mediante procesamiento de información.
